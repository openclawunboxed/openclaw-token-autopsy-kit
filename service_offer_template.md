# service offer template

## offer name

openclaw cost autopsy

## promise

find the hidden token leaks in your stack, cut waste, and leave behind a cleaner routing and scheduling model.

## what is included

- spend ledger audit
- heartbeat risk review
- cron migration recommendations
- context weight scan
- before and after verification
- written report
- one implementation handoff call

## what is not included

- model fine-tuning
- new workflow design from scratch
- managed hosting
- provider billing disputes

## good fit

- openclaw bill higher than expected
- recurring tasks with weak visibility
- one or more premium models running on routine work
- operator wants proof, not guesses
