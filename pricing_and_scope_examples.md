# pricing and scope examples

## starter audit

price: $250 to $500

good fit:
- one agent
- one or two recurring jobs
- one config review
- one report

## operator audit

price: $750 to $1,500

good fit:
- multiple agents
- mixed cron and heartbeat usage
- route cleanup
- before and after baseline
- implementation support

## client system audit

price: $2,000+

good fit:
- hosted or client-facing setup
- cost attribution by workflow family
- report delivery
- handoff docs
- follow-up review window
